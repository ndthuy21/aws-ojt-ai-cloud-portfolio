# GitHub Upload Steps

Remote configured locally:

```text
https://github.com/ndthuy21/aws-ojt-ai-cloud-portfolio.git
```

## Option A: Create Repository On GitHub Website

1. Open GitHub.
2. Create a new repository named:

```text
aws-ojt-ai-cloud-portfolio
```

3. Keep it public.
4. Do not initialize with README, `.gitignore`, or license because this local
   repo already has those files.
5. Push from terminal:

```bash
cd /Users/ndthuy21092005/dev/DPL-PROJECT/aws-ojt-ai-cloud-portfolio
git push -u origin main
```

## Option B: Ask Codex To Push

After the GitHub repository exists, ask Codex:

```text
push repo aws-ojt-ai-cloud-portfolio lên GitHub
```

Codex can then run the push command from this local Git repository.
